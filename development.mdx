ORT is an open source project and welcomes contributions. See the [contributing guide](https://github.com/oss-review-toolkit/.github/blob/main/CONTRIBUTING.md) and [issue tracker](https://github.com/oss-review-toolkit/ort/issues) on GitHub.

## Build from source

<Steps>
  <Step title="Clone the repository">

```bash
git clone https://github.com/oss-review-toolkit/ort
cd ort
git submodule update --init --recursive
```

  </Step>
  <Step title="Build the CLI">

```bash
./gradlew :cli:installDist
```

The ORT binary is at `cli/build/install/ort/bin/ort`.

  </Step>
  <Step title="Run the tests">

```bash
./gradlew test
```

  </Step>
</Steps>

## Static analysis

ORT uses [detekt](https://detekt.dev/) for Kotlin static analysis. Run it with:

```bash
./gradlew detekt
```
